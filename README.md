# SCT_DS_2
# Titanic Dataset - Data Cleaning and Exploratory Data Analysis (EDA)

## Overview

This project was completed as part of the Data Science Internship at SkillCraft Technology.

The objective was to perform Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic Dataset and identify relationships, patterns, and trends among variables.

## Dataset

* Source: Kaggle Titanic Dataset
* Records: 891 passengers
* Features: Passenger information, age, gender, class, fare, and survival status.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Data Cleaning

* Handled missing values in Age and Embarked.
* Removed Cabin column due to excessive missing values.
* Checked for duplicate records.

## Exploratory Data Analysis

Visualizations created:

1. Survival Distribution
2. Survival by Gender
3. Passenger Class Distribution
4. Survival by Passenger Class
5. Age Distribution
6. Fare Distribution
7. Correlation Heatmap

## Key Findings

* Female passengers had a higher survival rate.
* First-class passengers were more likely to survive.
* Most passengers were aged between 20 and 40 years.
* Higher fare passengers showed better survival chances.
* Third-class passengers had the highest mortality rate.

## Conclusion

EDA helped uncover important patterns and relationships in the Titanic dataset, demonstrating the significance of data cleaning and visualization in data analysis.
